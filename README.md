## 📦 Project Overview

**Dern-Support** is an IT support management system built using **Laravel** and **React.js**. It allows businesses and users to interact through a structured ticketing and support system — covering technical issues, user management, and internal workflows.

---

## 🚀 Features

- 🔐 **Authentication & Authorization** (Laravel Sanctum)
- 📝 **Support Ticket System** for managing client issues
- 📅 **Appointment/Task Scheduling**
- 📊 **Admin Dashboard** with service tracking
- 📁 **File & Media Handling**
- 📬 **Email Notifications**
- 💡 **Extensible & Modular Codebase**
